# Advent of Code 2020

Solutions for Advent of Code 2020 written in Python 3.14.

## Usage

```console  
./aoc-2020.sh [-h] [-f FILE] day  
```

### Positional arguments

`day`  
&emsp;&emsp;Day to answer. Must be an integer between 1 and 25.

### Options

`-h`, `--help`
&emsp;&emsp;Show a help message and exit.

`-f`, `--file FILE`
&emsp;&emsp;Read input from specified file.
